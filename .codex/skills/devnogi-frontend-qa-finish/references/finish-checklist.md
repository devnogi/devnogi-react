# Finish Checklist

## Visual consistency

- Match nearby spacing, radii, shadows, typography, and copy tone.
- Avoid isolated redesigns unless requested.

## Theme parity

- Check light mode and dark mode.
- Verify readable contrast and safe surface/text combinations.

## Responsive parity

- Check mobile, tablet, and desktop.
- Remove overflow and interaction traps.

## State coverage

- Check loading, empty, error, disabled, hover, focus, and selected states where relevant.

## Completion

- Run `npm run lint`.
- Fix introduced issues before sign-off.
