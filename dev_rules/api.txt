# 커뮤니티 화면
- 게시판 목록 조회 API: http://168.107.43.221:8080/dcs/api/boards
- 전체 게시판 게시글 목록 조회 API : http://168.107.43.221:8080/dcs/api/posts?page=1&size=20&sortBy=createdAt&direction=desc
- 게시판별 게시글 목록 조회 API: http://168.107.43.221:8080/dcs/api/posts/{boardId}?page=1&size=20&sortBy=createdAt&direction=desc
  - boardId는 '게시판 목록 조회 API' 사용 시 응답으로 받을 수 있음
